## buidler-boilerplate

### Install

```
$ yarn install && cp .env.example .env
```

Add your vars to .env after this

### Tests

```
$ yarn test
## or
$ yarn test:token
$ yarn test:counter
```

### Coverage tests

```
$ yarn test:coverage

## or

$ yarn test:coverage:token
$ yarn test:coverage:counter
```
